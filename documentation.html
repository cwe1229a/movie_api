myFlix Application

myFlix is a web application that grants users to obtain information about various pirate movies like their directors and genres. Users will also be able to sign up, update personal info, and creat a list of their favorite pirate movies

Features of the app

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get list of all pirate movies /movies Get n/a JSON object holding all data on pirate movies
Get data about one movie by title /movies/[title] Get :Title JSON object holding data about single pirate movie, contains director and title ex { Title: 'The Crimson Pirate', Director: 'Robert Siodmak' }
Get data about one movie by director /movies/[director] Get :Director JSON object holding data about single pirate movie, contains director { Director: 'Robert Siodmak' }
Let users register /users/register Post :Register Data containing new user details ex { name: '', favoriteMovies: '', } }
Let users update their information /users/[username] Put :Users JSON containing user updates to old information Text response that changes were successfully made
Let users add movies to their favorite list /favorites/add/[username]/[title] Post :Add JSON data containing new data from chosen movie to be added
Let users remove movies from theri favorite list /favorites/delete/[username]/[title] Delete :Username, :Title List is without selected movie that was removed
Let users unregister /users/unregister/[username] Delete :Users Notification/text that user was unregistered
